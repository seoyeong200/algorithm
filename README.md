# Greedy algorithm

- 지금 가장 최적인 답을 근시안적으로 택하는 알고리즘, 현재 상황에서 지금 당장 좋은 것만 고른다.
- 관찰을 통해 탐색 범위를 줄이는 알고리즘
- 정당성 분석이 중요하다. : 단순히 지금 가장 좋은 방법을 반복해서 선택해도 최적의 해를 구할 수 있는지 검토하기.
